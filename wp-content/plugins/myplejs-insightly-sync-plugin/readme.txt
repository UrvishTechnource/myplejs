=== myplejs Insightly Sync Plugin ===

------Description------

To post this :

{  
   "new_user_company":"TET",
   "new_user_role":"uthyrare",
   "new_user_last_name":"Ahmad",
   "new_display_name":"waqiamu@gmail.com",
   "new_user_email":"waqiamu@gmail.com",
   "user_id":"10457",
   "new_user_first_name":"Waqar",
   "old_user_email":"waqiamu@gmail.com",
   "old_display_name":"waqiamu@gmail.com"
}

type of payload to a web service url (GAS) from where it is used to sync contacts with Insightly.
